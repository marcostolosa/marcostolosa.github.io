Capabilities

Everything you need to map and drive the Windows API

From offline lookups to live execution — a single tool that removes the browser round-trips from your reversing workflow.

Structured documentation

Signature, parameters, return values and remarks scraped from Microsoft Learn into clean, parseable output.

Live WinAPI execution

Call any exported function via ctypes with typed returns, buffer allocation, hexdumps and GetLastError reporting.

61k+ mapping database

A massive function→header index with heuristic classification resolves APIs even when search comes up empty.

Multiple output formats

Rich terminal, JSON and Markdown output — all localized — so results drop straight into scripts and notes.

English & Portuguese

Full bilingual docs with -l br — signatures, parameter tables and remarks all translate together.

Fast & cached

Async HTTP with ETag conditional caching and a self-learning URL cache keep repeat lookups near-instant.

Two modes, one binary

Look it up, then run it

Documentation mode is cross-platform. Execution mode runs the real API on Windows.

Documentation mode

cross-platform

Resolve any Win32, NTAPI or driver function to its full signature and reference — no browser required.

$ manw-ng CreateProcessW $ manw-ng -l br NtCreateFile --tabs $ manw-ng RegOpenKeyExW -o markdown

Execution mode

windows-only

Invoke live APIs with module shortcuts, typed returns, writable buffers and error inspection.

$ manw-ng exec u:MessageBoxW 0 "Hi" "MANW-NG" 0 $ manw-ng exec u:GetCursorPos '$b:8' $ manw-ng exec k:OpenProcess 0x0400 0 4242 --show-error

Under the hood

A 5-step lookup pipeline

Each request cascades through fallbacks until an authoritative match is found.

1

Learn Search

Queries the official Microsoft Learn Search API first.

2

Smart URLs

Generates and validates candidate doc URLs from patterns.

3

Classifier

Heuristic header classification maps the function name.

4

Catalog

Falls back to the bundled internal API catalog.

5

Offline map

Resolves from the 61k+ offline JSON mapping as last resort.

Get started

Up and running in seconds

Clone, install, and you get the manw-ng console command.

$ git clone https://github.com/marcostolosa/manw-ng.git && cd manw-ng && pip install -e .
  • 01 Requires Python 3.11+ and pip.
  • 02 Documentation mode runs anywhere — Linux, macOS, Windows.
  • 03 Execution mode needs a native Windows Python interpreter.
